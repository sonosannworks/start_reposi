# Contactページ：ガイドブック紹介ボックス改修指示書

## 目的
contactページ内、おたすけガイドブック（お金編）の3つの個別ボックス（傷病手当金編／障害年金編／退職編）について、以下を変更する。

1. 現在の絵文字アイコン（📔🦽📋など）を、表紙画像（バナー形式の画像）に置き換える
2. 画像の下にタイトルを表示する
3. 購入ボタンの**上**に「はじめにを見る」ボタンを追加し、クリックで開閉するアコーディオンで「はじめに」本文を表示する

## 事前調査（必ず最初に行うこと）

contactページのHTML/JSを確認し、3つのボックスが

- (A) HTMLに直接3つ書かれている（静的）
- (B) JSの配列・テンプレートで動的生成されている

のどちらかを判断してから、該当する方の実装方法で進めること。配列で管理されている場合は、配列のデータ構造に `image`, `title`, `intro`（はじめに本文）のフィールドを追加する形で実装する。

## 画像ファイル

3枚の表紙画像（横長バナー、1870×841px、約2.22:1）を `images/` フォルダ（新規作成）に配置する。

| ファイル名（配置後） | 対応する本 |
|---|---|
| `images/cover-shobyo.png` | 傷病手当金編 |
| `images/cover-shogai.png` | 障害年金編 |
| `images/cover-taishoku.png` | 退職編 |

※画像ファイル自体は別途人間がアップロードする。Claude Codeは配置パスとファイル名の参照のみ実装すればよい（ファイルが見つからない場合はその旨を報告する）。

## レイアウト仕様（各ボックス内、上から順）

1. **表紙画像**
   - `width: 100%`
   - `aspect-ratio: 1870 / 841` を指定し、画像比率を保ったまま枠を確保（レイアウトのガタつき防止）
   - `object-fit: contain`（画像全体をそのまま見せる。トリミングしない）
   - 角丸はカード上部に合わせる（カード自体に `overflow: hidden` を設定）

2. **タイトル（テキスト）**
   - 画像のすぐ下に配置
   - 既存の見出しスタイルに合わせる（フォントサイズ・太さは既存の `<h3>` 等のクラスを流用）
   - タイトル文言：
     - 傷病手当金編 → 「傷病手当金申請ガイドブック」
     - 障害年金編 → 「障害年金申請ガイドブック」
     - 退職編 → 「退職後にもらえるお金ガイドブック」

3. **価格表示**（既存のまま、変更不要）

4. **「はじめにを見る」ボタン**（新規追加・購入ボタンの直上）
   - クリックで直下のテキストブロックを開閉（アコーディオン）
   - 開閉アイコン（シェブロンなど）を回転させて状態を示す
   - 初期状態は閉じている
   - ボタン文言：「はじめにを見る」／開いている時は「閉じる」に変えてもよい

5. **「はじめに」本文ブロック**（新規追加）
   - 初期状態 `display: none`
   - 開いたときに改行を保持して表示（`white-space: pre-wrap` または `<br>` 変換）
   - フォントサイズは本文より小さめ（13px程度）、色は既存の secondary テキストカラーを使用

6. **購入ボタン**（既存のまま、変更不要）

## 動作確認チェックリスト

1. 3つのボックスそれぞれに表紙画像が表示される（崩れ・引き伸ばしなし）
2. 画像の下にタイトルが正しく表示される
3. 「はじめにを見る」ボタンをクリックすると、対応する本文が展開される
4. 再クリックで閉じる
5. 購入ボタンの位置・機能は従来通り動作する
6. スマホ幅（375px程度）でレイアウトが崩れない
7. 画像が見つからない場合、コンソールエラーではなく代替表示（altテキストなど）になる

## 注意事項

- 既存の価格・購入ボタンのリンク（Stripe Payment Link）は変更しない
- 3冊セット（¥200）のボックスは今回の改修対象外（個別3冊のボックスのみ）
- 既存のCSS変数・クラス命名規則がある場合はそれに合わせる
